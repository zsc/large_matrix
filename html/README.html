符号约定

  • $\mathbf{A}, \mathbf{B}$：矩阵
  • $\mathbf{x}, \mathbf{y}$：向量
  • $\lambda_i$：特征值
  • $\kappa(\mathbf{A})$：条件数
  • $\mathcal{O}(\cdot)$：计算复杂度
  • $\mathbb{E}[\cdot]$：期望
  • $|\cdot|_F$：Frobenius范数
  • $\otimes$：Kronecker积
  • $\odot$：Hadamard积
